Game data services

Azure blob storage

Get cloud storage for game data, including all files types and unstructured data like images, audio, video, and more.

Cosmos DB

Use this low-latency game database to read, write, and query game and player data faster and with schema flexibility.

Azure Table storage

Store game data flexibly—according to its needs and function. For instance, store game transactions or changes in game states in temporary data storage tables.

Azure SQL Database

Scale your structured storage of relational data—like in-game transactions.

Azure Data Warehouse

Collect and analyze game data on a massive level, processing from both relational and non-relational databases.

Azure batch rendering

Render complex game graphics with the power of the cloud that integrates with your existing tools. Scale to thousands of cores easily – and only pay for what you use.
